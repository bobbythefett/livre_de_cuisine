# Livre de Cuisine

Dépôt centralisé pour les recettes au format [Cooklang](https://cooklang.org/).

## Structure
- `recipes/` : Dossier contenant les fichiers `.cook`.
- `GEMINI.md` : Instructions pour l'agent Gemini CLI.

## Utilisation
Utilisez l'extension Cooklang pour visualiser les recettes ou l'outil CLI Cooklang.
